# ▶ YouTube Summarizer

Gemini API を使って YouTube 動画を一瞬で要約するWebアプリ。

## 機能
- 📝 **要約** — 概要・主要ポイント・詳細・結論
- 🎯 **要点抽出** — 重要ポイントを箇条書き
- ✅ **アクション化** — タスクリストに変換
- ✏️ **カスタム** — 自由なプロンプト

## 使い方
1. Gemini API キーを入力・保存
2. YouTube URL をペースト
3. モードを選んで「要約する」

## Tech
- React 18 (CDN)
- Gemini API (YouTube URL直接入力 / preview機能)
- Vercel でホスティング
